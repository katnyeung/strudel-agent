# Changelog — Lo-fi DJ

## v2 (2025-03-08)
- **Complete rewrite: piano-first café aesthetic**
- Piano/keys now 60% of sonic space (was background at 0.2-0.35)
- Build order reversed: keys → hats → kick → bass → ghost snare
  (was kick → snare → hats → chord → bass)
- Added "Sonic Hierarchy" section with explicit percentage allocations
- Kick patterns simplified: "0,8" default (was "0,4,6,10,14")
- Kick gain reduced to 0.3-0.4 (was 0.4-0.6)
- Hat gain reduced to 0.08-0.15 (was 0.1-0.25)
- Snare gain reduced to 0.15-0.25 with degradeBy 0.35 (ghost only)
- Added evolution guidelines prioritizing harmonic changes over rhythmic
- Added example chord progressions (Classic, Dreamy, Moody)
- Piano gets reverb 0.5-0.7 + subtle delay for café depth
- Bass simplified to follow chord roots via <> brackets
- Included Rhythm Grid Rules from pattern debugging sessions
- Added "the piano alone must sound complete" design principle
- Removed four-on-the-floor from approved kick patterns
